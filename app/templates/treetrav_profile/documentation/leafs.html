Leafs Documentation

Leafs are simple HTML pages that enable users to create and post content written in Markdown directly to their profile. This feature offers a user-friendly and straightforward way to share information or stories. When creating a Leaf page, users are required to provide the following details:


Leafs Functionality

  • Folder Path: This is the path within the user's profile where they wish to post the Leaf page. The format for specifying the folder path must follow the structure of 'folder/subfolder/...'. Users can choose the destination folder that best fits the content they are creating.
  • File Name: The File Name serves as the title or description of the newly created Leaf page. It allows users to give a meaningful and concise name to their content, making it easily identifiable.
  • Markdown Content: Users can input the content of the Leaf page in Markdown format. Markdown is a lightweight markup language that provides a simple way to format text and include various elements like headings, lists, links, and more. Users can compose their content using Markdown syntax for easy and effective content creation.

Posting a Leaf Page

To create and post a Leaf page, users follow these steps:

  1. Enter the desired 'Folder Path' to specify where the Leaf page will be posted in their profile.
  2. Provide a descriptive 'File Name' to give the Leaf page a recognizable title.
  3. Compose the content of the Leaf page using Markdown in the 'Markdown Content' field.
  4. Click the "Post" or "Create" button to publish the Leaf page to their profile.

Once the Leaf page is posted, it becomes a part of the user's profile and can be accessed by other users who have access to the specified folder.

Leafs provide users with a straightforward way to create and share content within their treetrav profile. Whether it's sharing stories, articles, or information, Leafs allow for content creation in an easy-to-format and easily accessible format.
