General Documentation


Feed Page

The Feed Page is the central hub of your treetrav experience, providing a personalized view of your activity and connections. It offers several key features:

Post a Valid Link

Users can post a valid link by specifying the URL, adding a description, and selecting the folder path in which they want to save the link. The folder path must be in the following format: folder/subfolder1/subfolder2/.... This format allows for effective link organization and categorization.

Latest Feed

The Feed Page also displays the latest feed based on the users you follow. Users can view and interact with the links and content shared by those they follow, ensuring they stay up-to-date with the latest information.


Explore Page

On the Explore Page of treetrav, users can explore content and discover new posts made by other treetrav users. The Explore Page offers the following functionality:

View Latest Posts

Users can access and view the latest posts shared by treetrav users, regardless of their follow status. This feature is designed to encourage content discovery and facilitate connections with a broader user base. It allows users to stay updated on a wide range of topics and interests within the treetrav community.


Profile Page

The Profile Page is a user's personal space on treetrav, offering a comprehensive view of their profile and interactions. It encompasses several features:

View Personal Profile

Users can access their own profile, where they can see detailed information about their activity on treetrav. This includes any folders or links they have posted. The Profile Page provides an overview of a user's contributions to the platform.

User Details

In the User Details section, a user's profile picture, name, and the number of links they have posted are prominently displayed. These details help other users learn more about the profile owner and their contributions to the platform.

Followers and Following Lists

Users can access lists of their followers and the users they follow. This feature is essential for managing one's network and connections within the treetrav community. It provides a clear view of who is following the user and who the user is following, fostering a sense of community and interaction.

Additional Functionality

By clicking on the three dots in the top-right corner of the Profile Page, users can access additional functionality. These additional options may include the ability to edit the profile, adjust privacy settings, and manage notifications. This menu serves as a hub for various profile-related settings and actions.

Follow and Folder Share Requests

Any pending follow requests or folder share requests will be displayed in the Profile Page. Users can review and respond to these requests from this section, allowing for convenient management of social interactions and collaborative content sharing.
