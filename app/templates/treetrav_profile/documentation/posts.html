Posts Documentation


Posts

Posts are links that a user has shared on their profile. When creating a post, users specify the following details:

  • URL: The web address of the link they want to share.
  • Description (Optional): A text description of the link. If no description is provided, the post will display the link itself as the description.
  • Folder Path (Optional): The location in the user's profile where they want the link to be posted. If no folder path is specified, the link gets posted directly into the user's base profile page.

Interacting with Posts

Once a link has been posted, users with access to the post can interact with it in several ways:

  • Clicking the Icon or Link/Description: This action will open the respective link in a new tab, allowing users to explore the content linked to by the author.
  • Clicking the Author's Name: Users can click the author's name to navigate to that author's profile page. This feature provides an easy way to discover more about the author and explore their other posts and content.
  • Clicking the Path: Clicking the path associated with the post will take users directly to the exact location within the author's profile where that link has been posted. This feature provides quick navigation to the specific context of the shared link.
  • Clicking the Copy Icon: Clicking the copy icon will copy the link of the post to the user's clipboard, making it easy to share or save the link for future reference.
  • Clicking the Garbage Can Icon (Author Only): Users who are the author of the post can click the garbage can icon to delete the post completely. This feature allows authors to manage and remove their posts as needed.
  • Clicking the Star Icon (Non-Author Only): Users who are not the author of the post can click the star icon to add the post to their favorites list. This feature allows users to curate a list of their favorite posts from other users.

For more details about posts shared under shared folders, please refer to the "Shared Folder Documentation." Posts are a versatile feature of treetrav, enabling users to engage with shared content, personalize their experience, and maintain a well-organized profile.
