Shared Folder Documentation

The shared folder feature allows you to easily share folders with other users on the platform. This documentation provides an overview of how to share folders, manage shared folders, and handle share requests.


Sharing a Folder:

  1. Enter the folder path: Specify the path to the folder you want to share. For example:
    Folder Path: shopping/wish list
  2. Add user(s) to share with: Enter the username(s) of the users you want to share the folder with, separating them with commas. For example:
    Share with: username1, username2, username3
  3. Click the "Share" button to initiate the sharing process.

Viewing Shared Folders:

Shared folders can be categorized into two types: outbound shares and inbound shares.

  • Outbound Shares: These are folders that you have shared with other users. You can view the path from which you've shared the folder and the path to which it is mounted.
  • Inbound Shares: These are folders that other users have shared with you. You can view the path from which the folder is shared and, for inbound shares, you have the option to update the path to which the folder is mounted.

Accepting Share Requests:

When someone shares a folder with you, you will receive a share request. To accept the share and make the folder accessible in your profile, follow these steps:

  1. Specify a path to mount the folder to in your profile. This is the location where you want the shared folder to be accessible.
  2. Accept the share request.

Effect on Shared Content:

Once you've accepted a shared folder and mounted it to your profile, any posts you add within this path will be visible to all users with access to the shared folder. Similarly, if you delete a post within the shared path, it will be deleted for all users who have access to that folder.


Managing Shared Folders:

You have the flexibility to delete the shared folder at any time. This action will remove access to the shared folder for all users it was shared with.
